The Strategy — Maximize All 3 MJ Reference Slots

Every species gets a curated triad. Each slot does a different job — stop overlapping them.

Slot 1 · Reference Image
drag-drop / image prompt
Subject & composition. What the animal is and how it's posed. 1–2 max — more = collage bleed.
FEED IT → best full-body paleoart side profile + a clean skeletal mount
Slot 2 · Style
--sref
Texture, palette, light. Skin feel, teeth color, claw sheen. Ignores pose — stack 3–4 freely.
FEED IT → komodo / crocodile hide, teeth & claw macros, eye close-ups
Slot 3 · Omni
--oref  (ONE only)
Subject preservation. Locks the one canonical body plan. Must be photoreal — illustrations corrupt it.
FEED IT → the single best photoreal full-body restoration per species
