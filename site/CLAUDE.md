# site/ — Astro frontend (site-custodian's domain)

Public-facing storefront at **https://jurassinkart.com**. Static Astro build, Vercel-deployed from `main`, auto-rebuilt on every push.

## Owner & scope

Owned by **site-custodian** subagent. This domain is responsible for:
- Astro components, pages, layouts
- Image pipeline (build-time AVIF/WebP via Astro `<Image>`)
- SEO surface: JSON-LD, OpenGraph, sitemaps, robots.txt
- Lightbox / gallery UX
- Wiring Printify product URLs into Buy CTAs

**Out of scope:** Midjourney prompts, reference images, DB, the watcher (`tools/`), Printify API itself.

## Drop folder UX (the contract with `tools/`)

User drops a `.png` into `site/src/assets/gallery/<category>/`. Within 10s:
1. launchd watcher fires `tools/sync_and_deploy.sh`
2. `tools/sync_gallery.py` regenerates `src/data/products.json`
3. Watcher commits + pushes to `main` → Vercel rebuilds → image is live with full SEO

**Critical:** images here are **regular git objects** (not LFS). The `.gitattributes` exclusion `site/src/assets/gallery/** -filter -diff -merge text=auto` is what makes Vercel builds work — see RECAP.md 2026-04-30 LFS fix.

## SEO contract

Every gallery image must produce, on the deployed site:
1. `<picture>` with AVIF + WebP srcsets at 400w/800w/1200w
2. Visually-hidden long-form `<figcaption>` for screen readers + Google
3. Per-piece `VisualArtwork` JSON-LD with full `ImageObject`, `creator`, `about` (scientific name + era), `offers` (Etsy URL)
4. Entry in `dist/sitemap-images.xml` with absolute `https://jurassinkart.com/_astro/<hashed>.webp` URL

Site-wide JSON-LD (`Organization` + `WebSite`) is in `src/pages/index.astro`.

## Cross-domain reads (allowed)

- `printify/printify_ledger.json` → for Buy CTA deep-links per piece. If absent, fall back to Etsy shop home.

## Cross-domain writes (forbidden)

Site-custodian **never** writes to `tools/`, `printify/`, `mj/`, `refs/`, or the DB. If Buy URLs are broken, that's a printify-publisher bug; flag and stop.

## How to verify a change before pushing

```bash
cd site && npm run build
ls dist/sitemap*.xml                                # must exist
grep -c '<image:image>' dist/sitemap-images.xml     # must equal gallery file count
```

Local dev: `npm run dev` → http://localhost:4321
