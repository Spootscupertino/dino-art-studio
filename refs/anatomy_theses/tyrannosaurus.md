# Tyrannosaurus rex Anatomy Thesis

## Purpose
This file defines the core anatomical standards used to judge T. rex reference images, generated images, and future LoRA outputs.

The goal is not "cool T. rex."
The goal is "recognizably plausible Tyrannosaurus rex."

## Core Thesis
A good T. rex image should show a massive, horizontally balanced, tail-counterweighted predator with accurate theropod posture, reduced two-fingered forelimbs, powerful digitigrade hindlimbs, and a believable living-animal body envelope.

## 1. Horizontal, tail-balanced posture
- Spine should read mostly horizontal, not upright like old monster-movie reconstructions.
- Tail should be lifted and functioning as counterbalance.
- Reject tail-dragging, kangaroo posture, or vertical Godzilla stance.

Source basis: modern paleontological consensus favors active, alert, tail-off-ground posture with vertebral column close to horizontal.

## 2. Two-fingered reduced forelimbs
- Each hand should have two visible functional fingers.
- Arms should be short, not raptor-like, human-like, or three-fingered.
- Hands should not dominate the silhouette.

## 3. Powerful digitigrade hindlimbs
- Feet should carry weight on toes, not flat human/bear feet.
- Hindlimbs should look load-bearing and muscular.
- Avoid broken ankles, plantigrade feet, extra toes, or soft cartoon paws.

Theropods are generally described as erect, digitigrade, striding bipeds.

## 4. Massive skull, neck, and torso balance
- Skull should be large and deep, but not floating unsupported.
- Neck should feel muscular and integrated into the shoulders.
- Body mass should look physically supportable, not shrink-wrapped or noodle-necked.

Witton emphasizes that T. rex soft tissue is reconstructed from skeletal evidence, phylogenetic proxy, and functional prediction, so muscle bulk and body envelope matter.

## 5. Living animal, not movie monster
- Avoid exposed-teeth monster face unless the pose demands it.
- Avoid zombie shrink-wrap.
- Skin/integument can vary, but should feel biologically plausible.
- Prioritize animal realism over horror design.

Witton's framing is useful here: treat T. rex as a real animal, not just a celebrity monster or pop-culture icon.

## Scoring Categories
Each image gets scored 1–5:

1. Posture and balance
2. Hands and forelimbs
3. Feet and hindlimbs
4. Skull/neck/body mass
5. Living-animal realism

## Auto-Reject Conditions
Reject image if it has:
- Three-fingered hands
- Tail dragging on ground
- Upright kangaroo posture
- Flat human-like feet
- Broken or melted legs
- Tiny weak tail
- Shrink-wrapped skull/body
- Jurassic Park clone look as the main reference style

## Working Rule
If an image is beautiful but fails the thesis, it is not training data.
